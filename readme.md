# Runesmith v0.0.0
Compile html using templates and imports
---
## Install
---
```
npm install runesmith
```
## Usage
---
```
const Runesmith = require('runesmith');

Runesmith.compile();
```
Expand Runesmith's vocabulary by creating words of power
```
Runesmith.rune('gallery', file/path/to/gallery.html);
```
